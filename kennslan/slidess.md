# Markmið tímans

- Skipuleggja í 3 mismunandi skjöl
  - CREATE | INSERT | SELECT
- Færa verkefnið yfir í nýjan hreinan gagnagrunn
- **Endurbæta**
  - skorður (e. constraints)
- Skrifa flóknari select skipanir sem sýna fram á notagildi grunnsins

---

# Afhverju

- Betri skilningur
- Betri gagnagrunnur
- Læra af ""mistökum""

# Gátlisti

- Nafnavenjur (snake_case)
- Gagnatýpur
  - INTEGER | NUMERIC | REAL
  - CHAT | VARCHAR | TEXT
  - DATE | TIME | TIMESTAMP

---

## frh. Skorður

- NOT NULL
- UNIQUE
- CHECK (email LIKE '%@%') (price > 0)
- DEFAULT
- FOREIGN KEY
  - **ON DELETE** (RESTRICT | CASCADE)

---

# INSERT og SELECT

- Góðar (flóknar (ekki augljósar)) fyrirspurnir
- JOIN (2 töflur)
- GROUP BY (COUNT | SUM | AVERAGE)
